Gesamt Offline Calculation:

Pro Soldat für einen Rundgang berechnen:
    1. Netto Laufzeit (Abhängigkeit Bewegungsgeschwindigkeit) für Laufwege

    Dazu kommt:
       2.  Zeit für Mission
       3.  Zeit für Essen
       4.  Zeit für Klo
       5.  Zeit für Schlafen

        ->
        1 Soldat 1 Chair = Nur Essenszeit(Schnitt von allen Chairs)     // 10s
        2 Soldaten 1 Chair = EZ + ( 1 - 1/2 EZ)                         // 15s
        4 Soldaten 1 Chair = EZ + ( 1 - 1/4 EZ)                         // 17,5s
        4 Soldaten 2 Chair = EZ + (1 - 2/4 EZ)                          // 15s
        8 Soldaten 4 Chair = EZ + (1 - 4/8 EZ)                          // 15s
        8 Soldaten 1 Chair = EZ + (1 - 1/8 EZ)                          // 18s

Zeit von 1.-5. = 1 Rountrip ^= Missionsgeld
Geld pro Stunde = (3600s / Anzahl Roundtrips) * Missionsgeld

Gesamtgeld = Summe von allen GpS pro Soldat
Abzüglich gewisser % dann noch
Auzuzahlender Betrag, Maximalbetrag beachten
